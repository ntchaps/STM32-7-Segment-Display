# STM32-7-Segment-Display

Physical NucleoF446RE Connections:
	Segments:
	Nucleo A0 → 220Ω → 5641AS pin 11
	Nucleo A1 → 220Ω → 5641AS pin 7
	Nucleo A2 → 220Ω → 5641AS pin 4
	Nucleo A3 → 220Ω → 5641AS pin 2
	Nucleo A4 → 220Ω → 5641AS pin 1
	Nucleo A5 → 220Ω → 5641AS pin 10
	Nucleo PC2 → 220Ω → 5641AS pin 5
	Nucleo PC3 → 220Ω → 5641AS pin 3
	
	Digital Pins:
	5641AS pin 12 → Nucleo D2
	5641AS pin 9  → Nucleo D3
	5641AS pin 8  → Nucleo D4
	5641AS pin 6  → Nucleo D5
